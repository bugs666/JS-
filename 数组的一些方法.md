# Array对象

## 1.	push方法

```
在数组的末尾插入指定的N个(N>=1)值,并返回数组的最新长度，改变原始数组
const arr = [1,2,3];
const len = arr.push(4,5,6);//6
consoloe.log(arr)//[1,2,3,4,5,6]
```

## 2.	pop方法

```
删除并返回数组的最后一项，改变原数组
const arr = [1,2,3];
const len = arr.pop();//3
consoloe.log(arr)//[1,2]
```

## 3.	shift方法

```
删除并返回数组的第一项，改变原数组
const arr = [1,2,3];
const len = arr.pop();//1
consoloe.log(arr)//[2,3]
```

## 4.	unshift方法

``````
在数组的开头插入指定的N个(N>=1)值,并返回数组的最新长度，改变原始数组
const arr = [1,2,3];
const len = arr.unshift(4,5,6);//6
consoloe.log(arr)//[4,5,6,1,2,3]
``````

## 5.	reverse方法和sort方法

```
reverse方法，反转数组[1,2].reverse() ==> [2,1]
sort方法，根据排序规则进行数组排序，该方法接收一个比较方法，比较方法接收两个参数，如果第一个参数应该排在第二个参数之前，就返回负值，排在第二个参数之后就返回正值，两个参数相等，就返回0
[8,4,6,1,2,7,3,5].sort((v1,v2)=>v1 - v2)===>[1,2,3,4,5,6,7,8]
[8,4,6,1,2,7,3,5].sort((v1,v2)=>v2 - v1)===>[8,7,6,5,4,3,2,1]
```

## 6.	slice方法和splice方法

```
slice方法，剪切数组，接收两个参数，剪切开始位置start和结束位置end，返回数组中从start起到end-1的位置的数据构成的数组，该方法不影响源数组
[1,2,3,4,5,6,7,8].slice(0,5)====>[1,2,3,4,5]
```

```
splice方法，根据接收的参数来决定其功能，且会影响源数组
1.	删除元素，接收两个参数，删除的开始位置start和结束位置end，删除数组中从start起到end-1的位置的数据，并返回其组成的数组
[1,2,3,4].splice(0,2)===>[1,2]
2.	插入元素,接收三个参数，要插入的位置index，0和要要插入的值
[1,2,3,4].splice(0,0,8)===>[8,1,2,3,4]
3.	替换元素，接收>3个参数,开始位置start，要删除的元素个数，要替换的值（可以传入多个）
[1,2,3].splice(0,2,5,6,8)===>[5,6,8,3]
```

## 7.	concat方法

```
数组拼接，接收N（>=1）个值，并返回拼接后的数组，不会影响源数组
[1,2].concat([3,4],[5,6],[7,8])===>[1,2,3,4,5,6,7,8]
```
